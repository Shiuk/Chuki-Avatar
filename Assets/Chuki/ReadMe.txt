Instructions: 
! Make sure you install Poiyomi first!
1. Go to: https://github.com/poiyomi/PoiyomiToonShader/releases
2. Download Poiyomi Version 8.0
3. On Unity's Menu Bar, Click on "Assets -> Import Package -> Custom Package"
4. Locate the Poiyomi Unity Package you downloaded.


1. On Unity's Menu Bar, Click on "Tools -> Chuki Installer"
2. Follow along with the Installer.
3. Edit your Texture using Edit Texture 1.4_Asymetrical or Symetrical and save over the Body.PNG file located in the Chuki Folder.
4. Upload!
